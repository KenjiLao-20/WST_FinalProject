# 5 Minutes Till Dawn - Top Down Shooter

A retro-style top-down shooter where you survive 5 minutes against endless hordes of enemies. Level up, choose power-ups, and fight increasingly difficult enemies.

## Play Online
https://kenjilao-20.github.io/WST_FinalProject/team.html

## Controls

WASD - Move character
MOUSE - Aim
LEFT CLICK - Shoot

## Features

- 5-minute survival timer
- Level up every 15/25/35+ kills
- 15+ unique power-ups
- Enemies get stronger every minute
- 8-bit retro style

## Setup Instructions (Python Version)

1. Download and extract the ZIP file
2. Install Python from python.org
3. Install Pygame: pip install pygame
4. Run python main.py in the terminal
